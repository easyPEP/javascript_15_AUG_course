Basics Variablen

Variablen
Eine Variable besteht aus 3 Teilen
            
              var variablen_name = 'test';



Die Deklarierung
Das Schlüsselwort 'var' zeigt dem Browser an, dass jetzt eine Variable erstellt wird.




Die Benennung
Danach folgt der Variablen-Name. Diesen Namen kann man sich beliebig aussuchen.




Die Zuweisung
Mit dem '=' Zeichen wird der Variable ein beliebiger Wert zugewiesen. Der VariablenName gilt jetzt als Platzhalter für den zugewiesen Wert

Basics - Typen

In Javascript gibt es bestimmte Typen von Werten.

Diese kann mann sich einfach mit dem Browserbefehl 'typeof' anzeigen lassen

              
                typeof 123;
                // -> "number"

                typeof 'ein String';
                // -> "string"
Strings
            
              'ein String';
              'noch 1 String!!!';
Alles zwischen einfachen Anführungszeichen gilt als String und wird als ein einzelner Wert betrachtet. Zeichen wie Buchstaben, Zahlen und Sonderzeichen können darin verwendet werden.
Typen
Strings
            
              'ein ' + 'String' + '!';
              // -> 'ein String!'
Mehrere Strings können mit dem '+'-Zeichen zu einen einzelnen String zusammengefügt werden.
Typen
Numbers
              
                3;
                1.5;
Dezimalzahlen werden mit einem '.' getrennt. Nicht mit einem ','
USA! USA! USA!
Typen
Numbers
              
                3 + 12;
                // -> 15
                5 - 1.5;
                //-> 3.5
                3 * 3;
                // -> 9
                6 / 2;
                // -> 3
Mit Zahlen kann wie im Taschenrechner gerechnet werden.
Typen
Numbers
              
                'Ein String' + 5;
                // -> 'Ein String5'
                5 + '5';
                // -> '55'
                5 * '5';
                // -> 25
Will man Strings und Zahlen addieren kommen komische Sachen raus. :-/
Basics - Funktionen
Eine Funktion besteht aus mehreren Teilen.
Mit Funktionen kann Code in Blöcke zusammengefasst werden
                
                  function addieren(zahl1, zahl2) {
                    return zahl + zahl;
                  }
Basics - Funktionen
              
                function addieren(zahl1, zahl2) {
                  return zahl + zahl;
                }
Das Schlüsselwort 'function' zeigt dem Browser an, dass jetzt eine Funktion erstellt wird. 
                
                  function funktionsName(parameter1, parameter2, ...) {
                    // Code block
                    return parameter1 + parameter2;
                  }
Der Funktions-Name kann beliebig ausgesucht werden. Es ist aber hilfreich, sie so zu benennen, dass man weiss was sie tut. Hier wäre z.B. "addieren" ein guter Funktionsname. 
                
                  function funktionsName(parameter1, parameter2, ...) {
                    // Code block
                    return parameter1 + parameter2;
                  }
Die Parameter sind Werte wie Zahlen oder Wörter (strings), welche in der Funktion als Variablen zur Verfügung stehen. Auch diese kann man beliebig benennen. Hier wären z.B. "zahl1" und "zahl2" gute Namen. 
                
                  function funktionsName(parameter1, parameter2, ...) {
                    // Code block
                    return parameter1 + parameter2;
                  }
Der Bereich zwischen den geschweiften Klammern ist der "Code Block". Dort kann beliebig viel Code ausgeführt werden. Z.B. Rechenoperation. 
                
                  function funktionsName(parameter1, parameter2, ...) {
                    var ergebnis = parameter1 + parameter2;
                    return ergebnis;
                  }
Das Schlüsselwort 'return' gibt an welchen Wert die Funktion zurückgeben soll. Danach wird die Funktion beendet und der Wert rechts von return zurückgegeben. 
                
                  function addieren(zahl1, zahl2) {
                    console.log(zahl1);
                    console.log(zahl2);
                    var ergebnis = zahl1 + zahl2;
                    return ergebnis;
                  }

                  var addierteZahl = addieren(3, 5);
                  console.log(addierteZahl);
Eine Funktion wird benutzt indem man den Namen schreibt mit einer einfachen Klammer dahinter. Die Werte, welche der Funktion als Parameter übergeben werden, gibt man Komma-separiert innerhalb der Klammer an.
